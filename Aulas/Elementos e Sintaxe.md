- O **script** consiste em elementos ou peças que nos permitem criar um programa ou script.

- As variáveis são declaradas utlizando a palavra : ***var***.

- ***Console.log()***, para imprimir mensagens na tela/consola.

- Em JavaScript tanto podemos utilizar aspas duplas (" ") ou aspas simples (' ').

- A função é declarada com ***function***.

- Utilizar ***"==="*** significa igual a.

- Se tivermos uma lista de itens podemos utilizar um ***Loop*** para processar cada item individualmente.

- Método ***forEach*** é o que utilizamos para processar cada item.

- Um ***objeto*** geralmente inclui *propriedades* e também pode incluir *métodos* que são funções internas.
  - Os objetos podem ser personalizados para guardar dados específicos.

- Para acessarmos às propriedades de um objeto precisamos do nome do objeto seguido de um ponto. Ex.:
  - Quando escrevemos $carro.marca$, estamos a perguntar pela propriedade marca do objeto carro

- ***DOM*** ou *Document Object Model* é o modelo do documento.
  - Funciona como chamariz para a manipulação de dados.
  - Interpreta as páginas web.

- O método ***querySelector*** é para selecionar um elemento específico.

- Os ***Eventos e os Listeners*** são essenciais para interatividade.
  - Um **Evento** é por exemplo clicar num ponto específico ou quando a página é completamente carregada no navegador.
  - Os **Listeners** são os ouvintes desses eventos.
